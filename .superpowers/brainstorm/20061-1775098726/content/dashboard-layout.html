Dashboard behaviors

  • Icon click on non-video page: Opens this dashboard in a new tab
  • Icon click on video page: Toggles the sidebar
  • Search: Tokenized, Chinese-aware, searches titles + summaries + annotations + tags
  • Card click: Opens the video on Bilibili in a new tab
  • Thumbnail: Uses stored og:image URL; falls back to colored gradient placeholder
  • Ratings shown: 👍/👎 for simple mode, ⭐ stars for detailed mode
  • Sort options: 最近观看, 添加日期, 标题, 标注数量
  • Filter options: By tag (multi-select chip picker), by rating type/value
